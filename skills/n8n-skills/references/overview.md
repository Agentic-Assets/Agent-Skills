# n8n Skills Reference Overview

## Resource Structure

This skill's detailed reference material is organized in the `resources/` directory rather than `references/`. The resources contain comprehensive documentation for 545+ built-in n8n nodes and 30 community packages.

## Quick Navigation

| Resource | Path | Content |
|----------|------|---------|
| **Node Index** | `resources/INDEX.md` | Complete index of all nodes by category |
| **Compatibility Matrix** | `resources/compatibility-matrix.md` | Node version compatibility |
| **Input Nodes** | `resources/input/` | Trigger and input node documentation |
| **Output Nodes** | `resources/output/` | Output and action node documentation |
| **Transform Nodes** | `resources/transform/` | Data transformation node documentation |
| **Trigger Nodes** | `resources/trigger/` | Event trigger node documentation |
| **Community Nodes** | `resources/community/` | Community-contributed node packages |
| **Guides** | `resources/guides/` | Workflow patterns and best practices |

## Usage

Start with `resources/INDEX.md` to find the specific node or pattern you need, then load the relevant resource file.
